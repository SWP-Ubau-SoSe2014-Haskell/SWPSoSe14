SyntacticalAnalysis

Description

License : MIT

Stability : stable

SyntacticalAnalysis receives output of Lexer and turns each rail function graph into a list of paths. Each path is a triple and contains a Path-ID, a list of lexemes and a Path-ID of the path that follows. Those lexemes are executed in order and sequentially.
